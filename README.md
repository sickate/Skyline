# Skyline
A minimal Event-Driven HTTP server which stolen from Cramp. (https://github.com/lifo/cramp)

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'skyline'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install skyline
